# Sparkbot

Sparkbot is a local-first AI workstation shell from SparkPit Labs.

The project is focused on private, self-hosted AI workflows for builders, hobbyists, and technical users who want a practical workspace for coordinating AI assistants, model providers, and guarded actions.

## Project status

Sparkbot is in early public v1.0.0 planning and bootstrap.

This repository is being prepared as the clean public release home for Sparkbot. Installation instructions, release artifacts, and stable package layouts will be added only after they are validated.

## Direction

Sparkbot is being designed around:

- A local-first workstation experience
- A Round Table workspace for multi-agent collaboration
- Provider setup and model configuration controls
- Private/self-hosted operation
- Guardian-gated actions for sensitive capabilities
- A clear path toward desktop packaging and public release testing

## Repository standards

This repository is maintained as a professional public project. Public documentation should describe only supported or planned public functionality and should avoid private infrastructure details, internal operating notes, or unsupported claims.
